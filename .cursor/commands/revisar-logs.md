# Revisar logs de Tomcat

Ejecuta el comando para mostrar las últimas líneas de los logs de Tomcat 6 y transtemare-web:

1. **Catalina** (logs del servidor): `C:\Program Files\Apache Software Foundation\Tomcat 6.0\logs\catalina.*.log`
2. **Errores de la app**: `C:\Program Files\Apache Software Foundation\Tomcat 6.0\log\transtemare-web-error.log`

Muestra las últimas 50 líneas de cada archivo y resume si hay errores relevantes.
